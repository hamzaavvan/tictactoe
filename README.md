## Requirements
1. NodeJS
2. NPM

## Usage
First, install the dependencies using `npm`
```bash
$ cd tictactoe
$ npm install
```

Start the server:
```bash
$ npm start
```

Or start server using node:
```bash
$ node server.js
```

Then, visit: http://localhost:3000/
